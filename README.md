# CS-LDRNet
The source code of our paper "Channel-specific Spatial Importance Map Generation and Lost-Detail Recovery via Asymmetric Gradient Projection for Infrared Small Target Detection", which will be available after the paper is accepted. 4_layers_128.rar contains the visualizations of channel-specific SIMs generated by the fourth-layer attention.
